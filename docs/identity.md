# natsx 身份

## 我是谁

`natsx` 是 FoundationX 的 **NATS 内部通信扩展模块**，提供 NATS 客户端封装和发布/订阅接口。

## 我做什么

- NATS 发布/订阅/请求-响应接口封装
- 连接管理和健康检查
- 消息编解码

## 我不做什么

- 不是业务消息路由 — 消息拓扑由调用方定义
- 不是模板源 — 模板生成属于 xlib-standard
- 不依赖其他存储模块

## 宪法合规

| 条款 | 遵循方式 |
|------|----------|
| §3.3 | 存储扩展，可依赖 kernel + observex (interface-only) |
| §3.4 | 不依赖 configx、业务域、其他存储扩展 |
| §1 P13 | 存储扩展之间平级协作 |
